Geofabric LDAPI

Application Programming Interface

This distribution of the Geofabric is presented as a Linked Data API which presents RESTful URI endpoints for each Register (list) of types of item and each individual item, such as a Catchment.

Different views and formats for each Register and individual item are accessible via either HTTP Content Negotiation or via the use of query string arguments (QSAs).

API URI patterns

This API uses the following patterns:

DATASET_URI/CLASS_TOKEN/(OBJECT_ID){0,1}?(_view=VIEW_TOKEN){0,1}(&_format=FORMAT_TOKEN){0,1}

In words: the dataset URI with an additional class token may be followed by an object ID and then the QSAs _view and _format may be appended with a VIEW_TOKEN or FORMAT_TOKEN respectively.

Example endpoints

  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchments') }}
    • register of Contracted Catchments, in the default format (HTML)
  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchments') }}?_format=application/ld+json
    • register of Contracted Catchments, in the JSON-LD format of RDF
  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchment', contractedcatchment_id=12101198) }}
    • a specific Contracted Catchment, #12101198
  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchment', contractedcatchment_id=12101198) }}?_format=text/turtle
    • a specific Contracted Catchment, #12101198, in the Turtle format of RDF

Views deliver information according to specification so:

  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchment', contractedcatchment_id=12101198) }}?_view=hyfeatures
    • a specific Contracted Catchment, #12101198, delivered according to the HY Features specifications
  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchment', contractedcatchment_id=12101198) }}?_view=hyfeatures&_format=application/n-triples
    • a specific Contracted Catchment, #12101198, delivered according to the HY Features specifications, in the N-Triples format

To understand what views and formats are available for any API endpoint, use the specialised view for that purpose, the Alternates View, so URI?_view=alternats e.g.:

  • http://linked.data.gov.au/dataset/geofabric{{ url_for('classes.contracted_catchment', contractedcatchment_id=12101198) }}?_view=alternates
    • the listing of all the views and formats available for a a specific Contracted Catchment, #12101198
