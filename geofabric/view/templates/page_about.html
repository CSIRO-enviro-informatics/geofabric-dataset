Geofabric Linked Data API

About

This is an Application Programming Interface (API) that delivers a Linked Data version of the Australian Hydrological Geospatial Fabric (the Geofabric)

It is being developed through a project involving CSIRO & Geoscience Australia to implement a series of national spatial Linked Data datasets for Australia. the project is called LOC-I: the Location Identifiers.

Much more information will be available here shortly as we continue to develop this tool!
