Geofabric Linked Data API

About

This is an Application Programming Interface (API) that delivers a Linked Data distribution of the Australian Hydrological Geospatial Fabric (the Geofabric).

This distribution of the Geofabric doesn't add any new information to the Geofabric other than associating data within it with classes and relationships within the HY Features ontology.

This tool is being developed through a project involving CSIRO & Geoscience Australia to implement a series of national spatial Linked Data datasets for Australia. the project is called LOC-I: the Location Identifiers.

This API is facilitated by the pyLDAPI tool, working on top of Python's Flask web framework, which, with custom Python code, reads Geofabric data from a Web Feature Service and a Web Map Service distributions of the Geofabric.
